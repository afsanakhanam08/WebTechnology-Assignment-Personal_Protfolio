# 🌐 Personal Portfolio Website

## 📌 Project Overview

This project is a responsive **Personal Portfolio Website** developed using **HTML5, CSS3, and JavaScript** as part of the **Web Technology Lab Assignment**. The main objective of this project is to present personal, academic, and professional information in a modern and user-friendly interface.

The portfolio showcases my educational background, technical skills, completed projects, and contact information. It follows a clean design with smooth navigation, responsive layouts, and simple animations to improve user experience.


# 🎯 Objectives

- To design and develop a professional personal portfolio website.
- To apply HTML, CSS, and JavaScript concepts learned in the Web Technology course.
- To create a responsive website that works on desktop, tablet, and mobile devices.
- To present academic achievements, projects, and technical skills in an organized way.
- To improve front-end web development skills.
